Help Center

  • mm cra -all

    This command checks all registered accounts.

  • mm ga "username"

    This command gets an account and returns username + password.

  • mm clear

    This command clears the terminal.

  • mm logout

    This command logs you out.

  • mm rpp

    This command resets the profile picture.

  • mm da "username"

    This command deletes a specific account.

  • mm da -all

    This command deletes all the registered accounts.

  • mm rta

    This command resets the temporary achievements. (win streak etc.)

  • mm ra "achievement"

    This command resets a specific achievement.

  • mm ra -all

    This command resets all achievements.
